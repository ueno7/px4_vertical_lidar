31 ### Setting Up PX4 SITL
32 See https://dev.px4.io/en/simulation/ros_interface.html.
33 
34 It is convenient to write settings in .bashrc and .profile.
35 
36 In Firmware directory,
37 ```bash
38 echo "source $(pwd)/Tools/setup_gazebo.bash $(pwd) $(pwd)/build/posix_sitl_default" >> ~/.bashrc
39 echo "export ROS_PACKAGE_PATH=\$ROS_PACKAGE_PATH:$(pwd)" >> ~/.bashrc
40 echo "export ROS_PACKAGE_PATH=\$ROS_PACKAGE_PATH:$(pwd)/Tools/sitl_gazebo" >> ~/.bashrc
41 echo "export GAZEBO_PLUGIN_PATH=\$GAZEBO_PLUGIN_PATH:$(pwd)/build/posix_sitl_default/build_gazebo" >> ~/.bashrc
42 ```

44 ### Exporting Gazebo Model Path
45 You need to set environment variable if you want to insert models in this pacakge from Gazebo GUI.
46 
47 In px4_simulation_stack directory,
48 ```bash
49 echo "export GAZEBO_MODEL_PATH=\$GAZEBO_MODEL_PATH:$(pwd)/Tools/sitl_gazebo/models" >> ~/.bashrc
50 ```
51 If you write these settings in .bashrc and .profile,
52 do not forget to source them for the first time.

54 ## Run Gazebo Remotely
55 
56 ### Export Environment Variables
57 In server machine, run
58 ```bash
59 export ROS_MASTER_URI=http://server_ip:11311
60 export ROS_IP=host_ip_server
61 ```
62 In client machine, run
63 ```bash
64 export ROS_MASTER_URI=http://server_ip:11311
65 export ROS_IP=host_ip_client
66 export GAZEBO_MASTER_URI=http://server_ip:11345
67 ```
68 
69 You can check host_ip of each machine by running
70 ```bash
71 hostname -I
72 ```

95 ## Notes about Launch Files
96 
97 ### hexa_j153.launch
98 In order to launch `hexa_j153.launch`, you need to put gazebo models of gazebo model database in your `GAZEBO_MODEL_PATH`.
99 Gazebo models of gazebo model database can be downloaded from https://bitbucket.org/osrf/gazebo_models/overview.
100 
101 This launch file needs Gazebo 7.4.0 or newer to launch because of [this](http://answers.gazebosim.org/question/18014/gazebo-7-ambulance-model-and-other-invalid-mesh-filename-extension-crash/) problem.
102 Installation of Gazebo from OSRF repository is described [here](http://gazebosim.org/tutorials?tut=install_ubuntu&cat=install).
103 Note that ROS Kinetic should be used with Gazebo 7.x series according to [this page](http://gazebosim.org/tutorials?tut=ros_wrapper_versions).
